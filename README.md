# Data_projects_TripleTen
Projects I completed while attending the TripleTen Business Intelligence Analytics Bootcamp.  
Each project applies real analytical workflows — from data cleaning and feature engineering to visualization, dashboarding, and executive‑level storytelling.

| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | [Manhattan Vacation Rental Market Analysis](#) | Analyzed Manhattan’s vacation rental market using advanced Excel techniques, including data cleaning, standardized feature engineering, and pivot‑based revenue modeling. Identified top‑performing neighborhoods, evaluated revenue patterns by bedroom count, and delivered an investment strategy prioritizing 1–2 bedroom units in high‑demand areas. |
| 2 | [E‑Commerce Conversion & Cohort Analysis](#) | Examined user behavior across the e‑commerce funnel and built monthly retention cohorts to understand engagement patterns. Identified the largest drop‑off between product views and cart opens, quantified conversion rates at each stage, and revealed a consistent decline in retention after month one. Provided recommendations to strengthen mid‑funnel movement and early‑stage re‑engagement. |
| 3 | [Superstore Performance & Recovery Strategy (Tableau)](#) | Built a multi‑page Tableau analysis to diagnose profit centers, loss‑makers, advertising opportunities, and return‑rate patterns. Identified high‑margin sub‑categories, products to discontinue, and state‑month combinations with the strongest advertising ROI. Delivered a recovery strategy focused on margin improvement, product rationalization, and targeted marketing. |
| 4 | [Superstore Returns Root‑Cause Analysis (Tableau Story)](#) | Developed a Tableau Story analyzing return behavior across customers, states, categories, and time. Identified high‑return customer segments, regional return hotspots, seasonal spikes, and sub‑categories driving disproportionate return activity. Provided targeted recommendations to reduce avoidable returns and improve operational efficiency. |
