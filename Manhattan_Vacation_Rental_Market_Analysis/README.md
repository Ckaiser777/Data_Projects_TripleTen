# Manhattan Vacation Rental Market Analysis

## Project Overview
This project analyzes Manhattan’s vacation rental market to understand which neighborhoods and property types generate the strongest revenue. Using advanced Excel techniques — including data cleaning, feature engineering, and pivot-based modeling — the analysis identifies high-performing areas, evaluates revenue by bedroom count, and highlights top-earning listings. The goal is to provide a scalable investment strategy based on real revenue patterns.

---

## Project Structure
- **Data Cleaning:** Standardized neighborhood names, normalized bedroom counts, recoded studio units, and created helper fields for analysis.
- **Feature Engineering:** Added `neighborhood_clean`, `bedroom_clean`, and a custom label for high-performing listings.
- **Pivot Analysis:**
  - Neighborhood attractiveness
  - Revenue by bedroom count
  - Revenue by neighborhood
  - Top-earning listings
- **Executive Summary:** Synthesized insights into a clear investment recommendation.

---

## Key Insights
- Midtown, Hell’s Kitchen, and the Lower East Side generate the highest monthly revenue.
- 1- and 2-bedroom units produce the strongest total revenue and offer the best availability.
- 6-bedroom units show exceptional average revenue but appear in only two neighborhoods.
- The top listing — a 2-bedroom in East Harlem — earns over $200K annually.
- The most scalable strategy is investing in 1–2 bedroom units in high-demand neighborhoods.

Excel (PivotTables, data cleaning, feature engineering, scenario analysis)
